Remove a branch that does not exist in remote.

## Install

```shell
$ pip install rmlb
```

## Usage

```no
usage: rmlb.py [-h] [-ro {-d,-D}] [-q]

optional arguments:
  -h, --help            show this help message and exit
  -ro {-d,-D}, --remove-option {-d,-D}
                        Options for removing a branch. The default is '-d'.
  -q, --quiet           No output.
```
